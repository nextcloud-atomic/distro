# Nextcloud Atomic build code

Repository for building [Nextcloud Atomic](https://nextcloudatomic.com) images.

## Build and run instructions (example for incus vm)

```sh
# Build image
mkosi -f

# Run incus vm from generated image
./run_incus_vm.sh
```
